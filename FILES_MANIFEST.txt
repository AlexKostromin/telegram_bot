БЫСТРЫЙ СТАРТ
==============================================

Вариант 1 - Скрипт (рекомендуется):
$ ./start-admin-panel.sh

Вариант 2 - Вручную:
$ cd admin_panel
$ python manage.py migrate
$ python manage.py createsuperuser
$ python manage.py runserver

Вариант 3 - Docker:
$ docker-compose -f docker-compose.admin.yml up
